# 🍕 Sistema de Fidelidade - Pizzaria La Romanna

## 📌 Descrição

Este projeto é um sistema de fidelidade para uma pizzaria, desenvolvido em Java.
Ele permite o cadastro de clientes, registro de compras e controle de pontos acumulados.

A aplicação foi desenvolvida inicialmente com interface gráfica (Swing) para demonstrar o funcionamento das regras de negócio.

---

## 🛠️ Tecnologias Utilizadas

* Java
* Swing (Interface gráfica)
* JDBC
* MySQL
* Git

---

## 📂 Estrutura do Projeto

```
src/
├── model        # Classes de dados (Cliente, Pedido)
├── dao          # Acesso ao banco de dados
├── service      # Regras de negócio
├── view         # Interface gráfica (Swing)
├── connection   # Conexão com o banco
```

---

## 🚀 Funcionalidades

✔ Cadastro de clientes
✔ Registro de compras
✔ Acúmulo de pontos
✔ Consulta de pontos
✔ Resgate de pontos

---

## ⚠️ Observação Importante

A interface Swing presente no projeto é apenas para demonstração das funcionalidades.

O front-end definitivo será desenvolvido separadamente por outra equipe.

---

## 🗄️ Banco de Dados

O sistema utiliza MySQL.

As tabelas principais são:

* Cliente
* Pedido

> A criação do banco está sendo realizada por uma equipe separada.

---

## ▶️ Como Executar

1. Clone o repositório:

```
git clone https://github.com/nortondefranca/Pizzaria-La-Romanna
```

2. Abra no NetBeans

3. Configure a conexão com o banco de dados na classe de conexão

4. Execute o projeto
